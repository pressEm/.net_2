Строка - это последовательная коллекция символов, которая используется для представления текста. String Объект - это последовательная коллекция System.Char объектов, представляющих строку; System.Char объект соответствует блоку кода UTF-16. Значение String объекта является содержимым последовательной коллекции System.Char объектов, которое является неизменяемым (т. е. доступно только для чтения). Дополнительные сведения о неизменности строк см. в разделе неизменность и класс StringBuilder . Максимальный размер String объекта в памяти составляет 2 ГБ или около 1 000 000 000 символов.
